## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Chat-Application.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Chat-Application
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start the Server

```bash
node index.js
```

### 5️⃣ Open Browser

```bash
http://localhost:3000
```

---

## 📡 How It Works

1. User connects to the server.
2. Socket.IO establishes a real-time connection.
3. Messages are sent instantly without page refresh.
4. Connected users receive messages in real time.
5. Server broadcasts messages to active clients.
