# 💬 Chat Application

A real-time chat application built using Node.js, Express, and Socket.IO that allows users to communicate instantly through a simple and responsive interface.

---

## 🚀 Features

- 🔹 Real-time messaging
- 🔹 Multiple users support
- 🔹 Responsive UI
- 🔹 Fast communication using WebSockets
- 🔹 User-friendly interface
- 🔹 Node.js backend
- 🔹 Express server integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Socket.IO
